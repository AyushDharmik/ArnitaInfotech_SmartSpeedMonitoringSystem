# Smart Speed Monitoring System for Road Safety

## Internship
Arnita Infotech – Embedded Systems Internship

## Project Overview
This project implements a Smart Speed Monitoring System using Arduino Uno and Infrared (IR) Sensors. The system measures the speed of a moving vehicle by calculating the time taken to travel between two IR sensors placed at a fixed distance.

If the measured speed exceeds a predefined speed limit, the system activates a buzzer and displays the vehicle speed on an OLED display.

## Hardware Used
- Arduino Uno
- 2 × IR Sensor Modules
- OLED Display (I2C)
- Active Buzzer
- Breadboard
- Jumper Wires
- USB Cable
